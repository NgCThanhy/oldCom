DOM Event

DOM Event là các sự kiện diễn ra trong DOM
=> nó sảy ra khi người dùng tương tác với trang wed hoặc bản thân trình duyệt wed hoạt động

    phân loại:

  • người dùng
    • The FocusEvent: khi tập trung (di chuột hoặc khối) thì phải
    • The InputEvent: khi nhập
    • The KeyboardEvent: khi tương tác bàn phím
    • The MouseEvent: khi tương tác chuột
    • The TouchEvent: khi tương tác cảm ứng
    • The WheelEvent: khi tương tác con lăn lên xuống
  • trình duyệt wed:
    như việc tải url hoặc reload....
  • ngoài ra còn một số event chung
