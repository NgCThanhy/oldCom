sử dụng bem trong css

  1. BEM

    là 1 tiêu chuẩn đặt tên lớp khi viết css

  2. ý nghĩa

      BEM :
    • viết tắt của Block Element Modifier
    • Block: khối (là khối cha bao quanh 1 đối tượng div á)
    • Element: thành phần trong khối (hình ảnh, link, chữ...)
    • Modifier: bổ sung (class mới) các thuộc tính cho Block và Element (backgrounf, color ....)

  3. lợi ích khi dùng BEM

    giúp thống nhất 1 cách đặt tên và dễ dàng quản lý khi làm việc nhóm

  4. cú pháp

    • .Block
    • .Block__Element
    • .Block--Modifier
    • .Block__Element--Modifier

  5. ưu - nhược điểm

      ưu điểm

    • tính rõ ràng
    • dễ tái sử dụng (do tách phần Modifier ra)
    • dễ hoạt động nhóm, dễ được debug
    • có tính module => ko bị ảnh hưởng chéo sang lớp khác

      nhược điểm

    • tên dài + rườm rà
    • giảm hiệu quả khi làm các dự án nhỏ
