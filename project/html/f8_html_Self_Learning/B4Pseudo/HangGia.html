Pseudo class

cấu trúc của một lớp giả là:
:< tên lớp giả >
lớp giả có tác dụng là truy suất đến một đối tượng cụ thể để thao tác trên đó. dưới dây là một số lớp giả chuyên dùng. nên google để tìm hiểu thêm

  • :root => sẽ tham chiếu đến phần tử gốc của file html => là thẻ < html>

    • vd như khi muốn tạo một biến bên trong thẻ html chẳng hạn

  • :hover => thẻ nào được tham chiếu đến sẽ kích hoạt khối lệnh bên trong khi có di chuyển chuột đến nó
  • :active => thẻ nào được tham chiếu đến sẽ kích hoạt khối lệnh bên trong khi nhấp chuột vào nó
  • :first-child => sẽ tham chiếu đến phần tử con đầu tiên của một thẻ
  • :last-child => sẽ tham chiếu đến phần tử con cuối cùng của một thẻ

Pseudo element

cấu trúc thì y như lớp giả nhưng dùng dấu ::
phần tử giả thì có thể dùng để tạo ra những phần tử hiển thị trên wed mà ko cần viết html mà chỉ cần viết css thui

  • ::before => thêm vào trước
  • ::after => thêm vào cuối
  • ::first-letter => ký tự đầu
  • ::first-line => đòng đầu
  • ::selection => khi chọn khối lại
